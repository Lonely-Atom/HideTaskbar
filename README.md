## 【重要说明】

   1. 此软件无窗口，运行后将自动收到任务栏托盘中。
   2. 隐藏任务栏后托盘也会被隐藏，所以请熟记以下快捷键，防止隐藏后无法恢复。
   3. 双击任务栏图标可以【隐藏/显示任务栏】。
   4. 在任务栏隐藏时，可使用快捷键将系统托盘呼出。

## 【快捷键】

   1. 隐藏/显示任务栏：【Ctrl + ~】
   1. 隐藏/显示系统托盘：【Alt + 1】
